Простота эксплуатации
Middleware Stalker

Middleware Stalker – бесплатное промежуточное ПО от Infomir, разработанное лучшими специалистами отрасли для легкого и бюджнтного запуска IPTV-проектов. Middleware Stalker предназначен для операторов и системных интеграторов, которые запускают проекты по предоставлению услуг IPTV, OTT и сервиса VoD на базе IPTV-приставок серии MAG.

Выгодное решение
Middleware Stalker – это лучшее решение для вашего бизнеса.
  • Бесплатно распространяемое решение
  • Открытый программный код
  • Постоянно развивающаяся платформа
Удобное управление контентом
Редактируйте список подключаемых ТВ-каналов, объединяйте их в тематические разде-лы, формируйте тарифные пакеты.
Предоставляйте пользователям такие удобные сервисы как ТВ-Гид (EPG), ТВ-архив (nPVR) и Машина времени (TimeShift).
С легкостью управляйте Видео по запросу. А для заполнения данных о фильме теперь достаточно просто указать его ID сайта Кинопоиск.
Легкий мониторинг и статистика
Получите полный контроль над устройствами, которые подключены к вашей сети. Сократите расходы на техническую поддержку абонентов, разрешая большинство возникающах проблем удаленно, без выезда к абоненту. Исчерпывающие статистические данные о популярности телеканалов и контента сервиса Видео по запросу позволят учесть интересы ваших абонентов при формировании пакетов услуг.
Гибкие тарифные планы
Вы создаете столько тарифных планов, сколько необходимо вашему бизнесу. Пакеты услуг могут включать в себя телевизионные каналы, функции ТВ-архива, TimeShift, премиум доступ к сервису Видео по запросу, а также к широкому спектру дополнительных сервисов предоставляемых Middleware Stalker.
